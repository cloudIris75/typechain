# Typechain

## About
TypeScript로 블록체인 만들기.

## Stack
Node.js, npm, TypeScript, tsc-watch, crypto-js

### Install
```
npm install typescript --save-dev
npm install tsc-watch --save-dev
npm install crypto-js
```

## Period
2022.1.17. ~ 2022.1.17. (2h 30m)
